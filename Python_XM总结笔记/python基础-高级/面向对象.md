什么是面向对象：
面向对象是相对于面向过程而言的，面向过程是根据业务逻辑依次编写代码，关注每一个流程的细节实现，而面向对象是在于用开发轮子和使用轮子的方式实现业务需求。
不都这样说么，想吃辣椒炒肉，然后面向过程就是买辣椒、买肉、买各种佐料，然后切辣椒、切肉，开火炒菜，盛菜，最后才实现我们的吃辣椒炒肉。
面向过程的话，不在乎制作的过程，要吃辣椒炒肉，掏出手机，打开美团外卖，下单，配送，开吃！
面向对象的概念比较抽象，面向对象由三部分组成：
封装、继承、多态
封装：定义：用类将一类具有相同特征和行为的事物封装起来；这里的特征指属性，行为指方法，事物指就就是我们对优势：对属性和方法可以进行权限设置；移植性和可拓展性强；就是将具体的实现逻辑封装起来，让使用者不去关注内部的实现，而只要了解实现的结果
	类与对象的概念里面包含类属性、类方法、实例属性、实例方法、静态方法，魔法方法，

继承：使用并拓展父类的功能来满足自己的具体业务需求	继承就涉及到方法的重写
多态：就是不同的对象调用同名的方法，实现的结果会不相同	python的话是动态语言 所谓动态语言就是变量使用前不需要声明它的类型 解释器会根据赋值来动态确定变量的类型 正是因为如此python中的多态是很弱化的
